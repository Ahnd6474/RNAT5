# RNAT5

세부사항: 
RNA 객체: AGUC의 파라미터화 된 좌표
파라미터: 여섯 개 backbone torsion(α, β, γ, δ, ε, ζ), base relative rotation인 χ, 그리고 sugar pucker 
메서드: 
get_coords: 
이전 residue의 frame/원자들 + 현재 residue의 torsion/pucker로 다음 residue 좌표를 결정하는 결정론적 forward kinematics
출력: RNA 분자의 좌표



트랜스포머 인코더: RNA 객체 서열(RNA 서열을 미리 객체로 바꿔야 함, 파라미터는 기본값 사용)을 읽고 멀티 셀프 어텐션을 적용, 각 RNA 분자의 상태는 알지 못함
입력: RNA 객체 서열(문자열X), 리간드 정보(여러개의 리간드 가능)
병렬 어텐션: RNA 객체만으로 진행, 가중 어텐션헤드(쌍별 헤드 보완), 리간드 어텐션 헤드
트랜스포머 디코더: 인코더의 정보를 기반으로 RNA 객체의 상태 산출(파라미터 업데이트) 
입력: 인코더가 출력(마스킹 X), 원본 서열, 인코더 디코더 어텐션, 병렬 셀프 어텐션
출력: RNA 객체 서열(상태를 업데이트)
해당 객체 서열을 다시 인코더에 집어넣어서 정제 가능

좌표 계산: 디코더가 산출한 상태들을 좌표로 변환
RNA 객체의 get_coords를 반복 적용하되 'RNA 분자가 붙을 좌표'는 이전 RNA의 인산 좌표를 사용한다. 


훈련 방식: 내가 가진것은 PDB,리간드 서열이니 거기있는 걸로 손실 구성
훈련 대상: 트랜스포머 인코더, 디코더

# RNAT5

세부사항: 
RNA 객체: AGUC의 파라미터화 된 좌표
파라미터: 적당히 정하자. 
메서드: 
get_coords: 
입력: RNA 분자와 그 상태, 'RNA 분자가 붙을 좌표'가 없을시 상대좌표를 출력
개략적인 구현: (RNA 분자가 붙을 좌표)+(RNA의 방향각과 이성질체과 이성질체를 고려한 분자간의 상대 좌표)
주의 사항: 시작과 종결 코돈이 시작하는 RNA 분자와 마지막 RNA 분자가 가지는 특이적인 부분을 보정해줌
출력: RNA 분자의 좌표

RNA 인코더: RNA 객체(상태를 포함한다)를 멀티 셀프 어텐션으로 인코딩해준다. 
입력: RNA 객체 서열
멀티 셀프 어텐션
출력: RNA 서열 임베딩

트랜스포머 인코더: 인코더로 서열을 읽고 멀티 셀프 어텐션을 적용, 각 RNA 분자의 상태는 알지 못함
입력: RNA 서열(문자열), 리간드 정보(여러개의 리간드 가능)
트랜스포머 디코더: 인코더의 정보를 기반으로 RNA 객체의 상태 산출(파라미터 및 어떤 이성질체인지) 
입력: RNA 인코더(이전시점까지의 디코더 출력), 인코더가 출력(마스킹 X), 출력(몇 번째걸 예측할지), 어떤 RNA 인지(AUGC 정도)
출력: RNA 객체(상태를 예측)

좌표 계산: 디코더가 산출한 상태들을 좌표로 변환
RNA 객체의 get_coords를 반복 적용하되 'RNA 분자가 붙을 좌표'는 이전 RNA의 인산 좌표를 사용한다. 

주의 사항: RNA 객체는 RNA의 상태를 포함한다. RNA 객체 서열은 RNA 객체의 나열이다. 그에 비해 RNA 서열은 문자열로 이루어지고 어떤 상태 정보도 포함하지 않는다. 

훈련 방식: Teacher Forced로 진행하고 손실은 각 산출한 좌표로 MSE 기준으로 한다. 
훈련 대상: RNA 인코더, 트랜스포머 인코더, 디코더
